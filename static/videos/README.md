# Dossier Vidéos

Placez vos vidéos publicitaires dans ce dossier :

- `video1.mp4` - Première vidéo publicitaire (affichée 3 secondes après le chargement de la page)
- `video2.mp4` - Deuxième vidéo publicitaire (affichée 3-5 secondes après la première)

## Format recommandé :
- Format : MP4
- Résolution : 720p ou 1080p
- Durée : 30-60 secondes maximum
- Taille : Moins de 10MB par vidéo

## Instructions :
1. Remplacez les fichiers `video1.mp4` et `video2.mp4` par vos propres vidéos
2. Assurez-vous que les noms de fichiers correspondent exactement
3. Les vidéos seront automatiquement affichées sur la page d'accueil
